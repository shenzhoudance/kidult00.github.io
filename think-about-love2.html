不要轻易说爱[下]

发表于 2011-02-09 | 分类于 叽歪 |

上一篇咱们说了为什么不要轻易说爱，但不代表就消极地拒绝去爱了，相反，严肃地对待，是因为更好地爱。

如果你总觉得生活少了一点什么，总觉得自己还可以做点什么，觉得自己还有那么一点点任性和理想主义，那么欢迎回到纠结的世界，继续旅程，寻找你的所爱。

Find it

找到能过一辈子的爱人容易么？很难。那么也就不要抱怨找到你的兴趣爱好、你的domain很难。

如果你从刚懂事开始就已经有自己的心头所好，就像莫扎特和他的音乐，就像阿甘和他的长跑，那赶紧感谢上天，你比别人节省了好多时间，生来就跟自己的soulmate在一起了，别人无法拆散你们。

当然，更大的机率是，你一开始并不确定你喜欢什么，那就开始寻找吧。

  • 什么东西能使你眼前一亮，让你wow一下？
  • 什么东西让你有冲动去做同样的事，并且觉得自己能做得更好？
  • 什么东西让你很想据为己有，最好是自己亲手做出来？
  • 什么事情你做起来会忘记时间忘记疲倦？
  • 什么事情让你觉得做了会很cool，很想亲身经历一下？
  • 什么事情会让你的一生与没有做这件事情完全不同？
  • …………

你一定曾经遇到过这些东西或事情，即便这些moment都很短暂，也常常伴随着各种消极的声音：“不务正业”“这没什么意义”“你以后用这个糊口吗”“你就不能想点实际的吗”…………一般来说，你都受不了这些质疑和盘问，因为那只是一点小小火花，经不起这种令人窒息的扑打。

但是，你最好想办法把这些火花小心地保留起来，制造一些机会，在没有人的时候，让你们单独相处，听听自己内心的声音（就像瞥见合眼的摔锅/mm了，别人说他很丑，你总得找个机会仔细看看吧？）。于是你进入了下一个阶段。

Test it

一见钟情其实蛮容易的，因为你不用付出什么。一旦要验证你是不是一时心血来潮，是不是虚情假意喜新厌旧，那就得严肃处理了。大多数情况，经不起考验的感情不会长久，虚假的“兴趣”“理想”都会在这个阶段见光死去。

标准很简单，跟爱一个人并无二致：你愿意么？

如果没有原因，就是喜欢，就是所有心思都在上面，就是做什么都愿意，那赶紧pass进入下一阶段。

当然，更大的机率是，你不确定啊，忽冷忽热，若即若离，烦心起来就想抛弃。别着急，这才是人之常情～大部分的感情是培养出来的，大部分的婚姻是需要经营的，荷尔蒙的作用并不能持续太久。

如果你准备跟一个人过一辈子，第一件事得去了解ta吧？所以，你得先透透彻彻地了解你感兴趣的东西，这绝对不轻松。如果越了解越感兴趣（大多数情况是这样的），欲罢不能，那你们进展不错。接着可以尝试为ta付出，付出时间、精力、情感，做一些很基本的事情（打基础啊，往往都是枯燥乏味直教人想放弃），看自己的热情能持续多久。

可以想象，这个阶段毁了多少好姻缘，因为大家耐不住寂寞，开始计较付出和回报。

要熬过这个阶段，其实是有技巧的。最简单的就是趁热打铁，用正浓的感情来淡化审美疲劳、单调无趣，等到感情进入平缓期，你们已经建立起了情感上的依附。例如，一段时间的疯狂已经让你小有成就感，或者养成了某种习惯。还有其他有用的招数：

  • 找那些比你更热爱、更专业的人指引你，例如找一个好的有激情的组织厮混；
  • 狠心投入大量的成本，让你欲罢不能，例如买一个很贵的网球拍，让你没有理由不去网球场。

很多时候，之所以不够喜欢，其实是投入得不够：不够深入、不够了解、没有共同的经历（尤其是痛苦的回忆）。如果你能做好一件事，你往往会喜欢上它。鉴于世界上真正的情圣并不多，所以请相信日久生情，请相信往往是先做好了再喜欢。

Go for it

能够过得了test这一关，你会发现其实你已经在go for it了～剩下的，就让爱和内心的声音指引你吧，all else will come。别忘了，旁人（包括你自己）会不停地表达他们的不解、担忧、嫉妒、愤怒、不屑、嘲讽等等各种情绪，你可以听一听，但大多数不用听进去，因为这些人大多数都太有空了————他们没有自己的所爱啊，只好通过指指点点给别人添堵来打发时间寻找乐子排解情绪了。

so

经过了三个阶段，这个时候，你应该有底气向其别人宣布：我喜欢xxx，我爱xxx，我热爱xxx。

Or，你根本不用表达，因为一旦真正发自内心地热爱了，与别人无关，你的眼中只有ta。

回想过去二十多年，自己不是没有喜欢的东西，但每一次，每一次都因为自己的懒惰和胆怯，没有珍惜那一次又一次的火花，没有拿出自己的勇气去爱，所以输给了平庸和无趣。花了20多年学到的这一课，代价很高：简单的才是强大的，真正的热爱没有借口。

希望不会太晚。其实，永远不会太晚。相见恨晚是一种很特别的美，很特别的情感。

希望，我能尽早确定我爱你，并且用余生来表达我的热爱
